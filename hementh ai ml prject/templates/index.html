        About the Project

        Civil Engineering AI/ML Mini Project

        This web application uses advanced Machine Learning techniques (Random Forest Regression) to accurately predict the 28-day compressive strength of high-performance concrete. By providing specific mix design proportions—including cement, blast furnace slag, fly ash, water, superplasticizer, coarse aggregate, and fine aggregate—engineers can rapidly prototype and validate concrete mixes without waiting for lengthy physical curing times.

        The underlying model was trained and rigorously validated on the renowned Yeh (1998) Concrete Compressive Strength dataset. It features built-in mix validation to check IS 456 standard water-binder ratios, automated Out-of-Dataset detection to warn users of extreme inputs, and real-time material estimation capabilities for practical site application.

        Why Dataset Ranges Matter

        This application was trained on real-world concrete mix data from the Yeh (1998) Concrete Dataset. Machine learning models are most accurate when the input values fall within the range of training data.

        If you enter values outside these ranges, the model is extrapolating beyond what it has learned, which may result in unreliable predictions. The Confidence Score and Out-of-Dataset warnings are automatically adjusted to reflect this.

        ⚠️ Always verify extreme predictions with physical cube testing.
